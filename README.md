# SmoothType 丝滑打字

## Chrome 安装教程

### 1. 下载项目

在 GitHub 项目页面点击：

```text
Code → Download ZIP
```

下载完成后，将 ZIP 压缩包完整解压。

也可以使用 Git 克隆项目：

```bash
git clone <你的 GitHub 项目地址>
```

### 2. 打开扩展管理页面

在 Chrome 地址栏输入：

```text
chrome://extensions/
```

然后按回车。

### 3. 开启开发者模式

打开扩展管理页面右上角的：

```text
开发者模式
```

### 4. 加载扩展

点击：

```text
加载已解压的扩展程序
```

选择解压后的项目文件夹。

所选文件夹中必须能够直接看到：

```text
manifest.json
content.js
popup.html
popup.css
popup.js
```

不要直接选择 ZIP 压缩包，也不要选择看不到 `manifest.json` 的外层文件夹。

### 5. 刷新网页

扩展安装完成后，刷新已经打开的网页：

```text
Ctrl + Shift + R
```

然后点击网页输入框测试平滑光标。

### 6. 打开设置面板

点击 Chrome 工具栏右侧的扩展图标，找到：

```text
SmoothType 丝滑打字
```

可以将它固定到工具栏，然后调整：

- 全局启用或关闭
- 当前网站启用或关闭
- 快速、Word、柔和三种模式
- 光标移动时间
- 光标宽度
- 光标闪烁周期

## Microsoft Edge 安装教程

### 1. 下载并解压项目

在 GitHub 项目页面点击：

```text
Code → Download ZIP
```

下载后完整解压 ZIP 压缩包。

### 2. 打开扩展管理页面

在 Edge 地址栏输入：

```text
edge://extensions/
```

然后按回车。

### 3. 开启开发人员模式

打开扩展管理页面中的：

```text
开发人员模式
```

### 4. 加载扩展

点击：

```text
加载解压缩的扩展
```

选择解压后的项目文件夹。

所选文件夹中必须能够直接看到：

```text
manifest.json
content.js
popup.html
popup.css
popup.js
```

### 5. 刷新网页

安装完成后，回到需要使用的网页并按：

```text
Ctrl + Shift + R
```

然后点击输入框测试。

### 6. 固定扩展图标

点击 Edge 工具栏右侧的“扩展”按钮，找到：

```text
SmoothType 丝滑打字
```

选择在工具栏中显示，即可随时打开设置面板。

## 更新扩展教程

下载新版本后，可以按照下面的方法更新。

### 方法一：重新加载原文件夹

1. 使用新版本文件替换原项目文件夹中的旧文件。
2. 打开 Chrome 的 `chrome://extensions/`，或 Edge 的 `edge://extensions/`。
3. 找到 `SmoothType 丝滑打字`。
4. 点击扩展卡片上的“重新加载”按钮。
5. 返回网页并按 `Ctrl + Shift + R` 强制刷新。

### 方法二：移除后重新安装

1. 在扩展管理页面移除旧版本。
2. 解压新版本。
3. 再次点击“加载已解压的扩展程序”或“加载解压缩的扩展”。
4. 选择新版本项目文件夹。
5. 刷新网页。

## 安装注意事项

- 扩展不能在 `chrome://`、`edge://`、浏览器扩展商店等受保护页面中运行。
- 安装扩展之前已经打开的网页通常需要刷新一次。
- 必须加载已经解压的文件夹，不能直接加载 ZIP 压缩包。
- 选择的文件夹中必须直接包含 `manifest.json`。
- 部分使用 Canvas 自绘输入区域的网站可能无法获得真实光标位置。
